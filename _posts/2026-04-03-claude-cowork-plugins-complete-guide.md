## What Are Claude Cowork Plugins?

Plugins are text-based instruction files that extend Claude Cowork with specialized workflows for your role. They're just markdown files that tell Claude how to behave in a specific domain — no code, no setup beyond a single click.

Anthropic open-sourced 11 ready-to-use plugins on GitHub, and you can install them directly inside the Claude Desktop app. They're free — the only cost is your Claude plan (available on Pro, Max, Team, and Enterprise).

---

## How to Install a Plugin in 5 Steps

Getting your first plugin running takes under two minutes:

1. Open the **Claude Desktop app**
2. Switch to the **Cowork tab** (top navigation)
3. Click **Customize** in the left sidebar
4. Click **Browse Plugins** to open the catalog
5. Click **Install** on any plugin — it's live immediately

That's it. No terminal, no config files, no API keys.

## Connect Your Existing Tools

Plugins can connect Claude to your existing tools via MCP connectors — pre-built integrations you authorize once, then Claude can read, search, and act inside them during your Cowork sessions.

- **Google Workspace** — Drive, Gmail, Google Calendar
- **Sales & CRM** — Apollo, Clay, Outreach
- **Finance & Research** — FactSet, MSCI, SimilarWeb
- **Legal** — LegalZoom, Harvey
- **Other** — DocuSign, WordPress

## Build Your Own Plugin (No Code)

Claude Cowork has a built-in **Plugin Create** tool that lets you build a custom plugin from scratch. Describe the workflow you want, and it asks you questions about your tools, standards, and processes — then generates the plugin file for you.

To customize any existing plugin, just edit its markdown file and add your company terminology, tools, and processes. That's all a plugin is under the hood.
