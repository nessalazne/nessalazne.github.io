## What Is Claude AI?

Claude is an AI assistant made by Anthropic. Unlike basic AI tools, Claude can hold long conversations, remember context, analyze documents, write code, and — when paired with Claude Code — actually execute tasks on your computer.

There are a few ways to use it:

- **Claude.ai** — the web interface, free to start
- **Claude Code** — the CLI tool that runs in your terminal and can take actions
- **Antigravity IDE** — a drag-and-drop visual environment that runs Claude Code skills with no terminal needed

For absolute beginners, start at [claude.ai](https://claude.ai). When you're ready to automate real workflows, move to Claude Code or Antigravity.

## 5 Things Claude Can Do Right Now (Without Any Setup)

You don't need to install anything to get value from Claude. Here are five tasks you can try today at claude.ai:

1. **Write a first draft** — paste a topic and say "write a 500-word blog post about [topic] for a beginner audience"
2. **Summarize a document** — paste in a long article or PDF text and ask "give me the 5 key takeaways"
3. **Rewrite your content** — paste a caption or email and say "rewrite this to sound more conversational and confident"
4. **Answer questions with sources** — Claude cites its reasoning; ask it to explain a concept step by step
5. **Build a system** — describe a workflow you do manually and ask Claude to design an automated version of it

These all work in the free tier. No credit card, no API key, no setup.

## How to Write Prompts That Actually Work

The biggest mistake beginners make is treating Claude like a search engine. Claude responds to *instructions*, not keywords.

**The 3-part prompt formula:**
1. **Role** — who Claude should act as ("You are a social media strategist")
2. **Task** — what you want it to do ("Write a week of Instagram captions")
3. **Context** — the details it needs ("The brand is a fitness coach targeting busy women over 30")

**Example prompt:**
> You are a content strategist for a creator brand. Write 5 Instagram captions for a reel about the benefits of AI tools for entrepreneurs. Keep each caption under 150 characters. Use a conversational, empowering tone. End with a CTA.

That level of specificity is what separates useful output from generic filler.

That's where **Claude Code** comes in. It's a CLI tool that gives Claude access to your file system, terminal, and browser. You can ask it to:
